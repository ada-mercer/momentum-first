# GR Primitive Source Axial K0 Reduction (v1)

Status: derivation note / figure support
Purpose: give the clean reduced radial field used by the primitive-source figure with a field that decays at infinity.

## 1) Motivation

The strict 2D point-source model uses

\[
\Delta_2 \ln r = 2\pi\delta^{(2)}(\mathbf x_\perp),
\]

so a nonzero integrated 2D source produces a logarithmic field. This is useful pedagogically, but it does not decay to zero as `r -> infinity`.

To keep the source interpretation clean while obtaining a decaying radial field, use a source that is still pointlike in the transverse plane but has finite axial extent.

---

## 2) Axial exponential source

Let

\[
f_{r_0}(z)={e^{-|z|/r_0}\over 2r_0},
\qquad
\int_{-\infty}^{\infty} f_{r_0}(z)\,dz=1.
\]

Use the transverse radial delta convention

\[
\delta^{(2)}(\mathbf x_\perp)={\delta(r)\over 2\pi r}.
\]

For the single-channel case,

\[
J_k^+(r,z)=Q_+{\delta(r)\over2\pi r}f_{r_0}(z),
\qquad
J_k^-(r,z)=0.
\]

The integrated source is still finite:

\[
\int J_k^+\,d^2x_\perp dz=Q_+.
\]

The balanced comparison at matched total source amount is

\[
J_k^+(r,z)=J_k^-(r,z)=
{Q_+\over2}{\delta(r)\over2\pi r}f_{r_0}(z).
\]

---

## 3) Midplane radial kernel

The 3D Green kernel for a point source is proportional to

\[
{1\over\sqrt{r^2+(z-z')^2}}.
\]

At the midplane `z=0`, convolve with the axial exponential profile:

\[
I_{r_0}(r)=
\int_{-\infty}^{\infty}
{e^{-|z|/r_0}\over 2r_0\sqrt{r^2+z^2}}\,dz.
\]

By evenness,

\[
I_{r_0}(r)=
{1\over r_0}\int_0^\infty
{e^{-z/r_0}\over\sqrt{r^2+z^2}}\,dz.
\]

Using the standard identity

\[
\int_0^\infty {e^{-az}\over\sqrt{z^2+b^2}}\,dz=K_0(ab),
\qquad a>0,\;b>0,
\]

with `a=1/r_0` and `b=r`,

\[
\boxed{
I_{r_0}(r)={1\over r_0}K_0(r/r_0).
}
\]

This gives the desired reduced profile.

---

## 4) Field expressions

For the single-channel source, the corresponding midplane field profiles are proportional to this kernel. With the coefficient convention inherited from the 3D point-source note,

\[
\theta_0(r,0)=
{G_N\lambda_M\over2c^2}{Q_+\over r_0}K_0(r/r_0),
\]

\[
\theta_i(r,0)=
{4G_N\lambda_P\over c^3}{Q_+\over r_0}\hat e_{k,i}K_0(r/r_0).
\]

Using the weak-field convention `A_i=-theta_i`,

\[
A_i(r,0)=
-{4G_N\lambda_P\over c^3}{Q_+\over r_0}\hat e_{k,i}K_0(r/r_0).
\]

For the balanced pair with matched total source amount,

\[
\theta_0^{\rm bal}(r,0)=
{G_N\lambda_M\over2c^2}{Q_+\over r_0}K_0(r/r_0),
\]

while the odd sector cancels:

\[
\theta_i^{\rm bal}(r,0)=0,
\qquad
A_i^{\rm bal}(r,0)=0.
\]

---

## 5) Asymptotics

Near the core,

\[
K_0(x)\sim -\ln(x/2)-\gamma,
\qquad x\to0.
\]

So the reduced field retains the familiar logarithmic core behavior.

Far from the source,

\[
K_0(x)\sim\sqrt{\pi\over2x}e^{-x},
\qquad x\to\infty.
\]

So the field decays to zero rather than having a strict 2D logarithmic tail.

---

## 6) Figure-use summary

The figure uses the replacement

\[
\ln(r/r_0)\quad\leadsto\quad K_0(r/r_0)
\]

as a clean reduced profile generated by finite axial structure.

The conceptual point remains the same as the strict 2D example:

- single-channel source: scalar + odd/shift response,
- balanced coincident pair: scalar response remains, odd/shift response cancels.

The axial profile only improves the radial behavior so the fields decay at large radius.
