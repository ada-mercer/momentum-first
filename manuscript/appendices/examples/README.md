# Examples Appendix

This folder holds educational worked examples and support notes that show how M1 notation is used in familiar cases.

These files are appendix/support material, not part of the main chapter spine unless explicitly listed in `_quarto.yml`.

Use this folder for:
- small calculation examples,
- notation-orientation examples,
- pedagogical translations of standard physics into M1 language,
- application sketches that are useful but not central enough for a main chapter.

Keep heavier algebraic derivations in `manuscript/appendices/derivations/`.
